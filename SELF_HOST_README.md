# Self-hosting the bot

## Windows quick start

1. Make sure Python 3.12+ is installed.
2. Double-click `start_bot.bat`.

The script loads `DISCORD_TOKEN` from `.env`, installs `requirements.txt`, then runs `main.py`.

## Keep it online

Keep the terminal window open. Closing it turns the bot off.

## Token safety

Do not upload `.env` to GitHub. It is ignored by `.gitignore`.
